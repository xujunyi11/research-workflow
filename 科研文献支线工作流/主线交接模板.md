# 主线交接模板

## 文献问题

```markdown
---
document_type: literature_question
version: v001
created_at: YYYY-MM-DD
based_on_idea: IDEA_vNNN 或讨论状态
domain_overview_revision: rNNN
---

# 文献问题_v001

## 1. 当前研究问题与原因假设
## 2. 需要作出的一个主要判断
## 3. 需要核查的论文/任务/机制范围
## 4. 重点比较的证据、反证和协议
## 5. 明确不需要覆盖的范围
## 6. 不同答案将如何改变 Idea
```

## 文献地图

文献地图与问题使用相同版本号。

```markdown
---
document_type: literature_map
version: v001
created_at: YYYY-MM-DD
based_on_question: 文献问题_v001.md
knowledge_revisions: [领域概览及主题知识修订号]
literature_cutoff: YYYY-MM-DD
---

# 文献地图_v001

## 1. 直接结论
## 2. 支持证据与证据强度
## 3. 反证、协议差异与替代解释
## 4. 最近邻：真正相同与不同之处
## 5. 对原因假设、创新边界、机制和最小实验的影响
## 6. 仍会影响决策的未知问题
## 7. 关键来源与证据定位
```

文献地图的价值在于降低关键不确定性；允许结论为“证据不足以改变当前判断”。
